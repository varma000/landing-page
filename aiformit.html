// faq

Questions engineers ask

Do I need to be a beginner at coding? +
No — this assumes you already code and use a CLI. It's built for working engineers, not first-timers.
Will I get a real pipeline in my portfolio? +
Yes. By Week 6 you'll have a self-healing ops pipeline (detect → diagnose → patch) with a live dashboard you can show any team.
Is this about agents or just scripts? +
Both. You'll build practical automations first, then layer agentic patterns (LangChain, tool-calling) on top by the capstone.
How is this different from the Beginners course? +
Beginners is a no-code first step. This course is code-first, tool-deep, and assumes engineering experience — it builds production systems, not concepts.
